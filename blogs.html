1. What is DOM and what is the purpose of DOM?

Ans : The Document Object Model (DOM) is a programming interface for web documents. DOM is an interface that defines how the browser reads html document. it represents the page so that programs can change the document structure, style and content. DOM manipulation is when you use javascript to add, remove and modify elements of a website.

2. How will you select HTML elements using DOM? Name the DOM methods.

Ans. There are 5 ways in which you can select elements in a DOM using selectors

  • getElementsById()
  • getElementsByClassName()
  • getElementByTagName()
  • querySelector()
  • querySelectorAll()
